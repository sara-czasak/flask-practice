So I'm trying to learn flask now

Flask is a really useful python library to help create dynamic web pages

You can use flask to serve up different pages based on the extension. For instance this file is called index.html and using render_template I can have app.py serve it up nice and easy. The key is to make sure all the html pages are saved in a subdirectory called templates because that's where flask looks for pages to be rendered with render template


Now lets pass some variables dynamically into this file

I went ahead and added a list in app.py called skills and then passed it on to this page using render_template. Let's make it show up here as an HTML unordered list shall we? We'll use Jinja for this

    {% for i in skills %}
  • {{i}}
    • {% endfor %}

Another thing you can do with Jinja, and mind you it works the same way as loops is conditional statements. I passed over a variable called is_logged_in and set it to False. With a little if statement we'll go ahead and display you're logged in or please log in. in this case if everything works correctly we should see plase log in since the variable is set to False:

{% if is_logged_in %}

You're logged in

{% else %}

Please log in!

{% endif %}
